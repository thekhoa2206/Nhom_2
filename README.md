# Nhom_2
Đồ án môn học: quản lí chuỗi khách sạn

Cách chạy project:

-Server: 
  Cách 1: Chạy trên intellij

  Cách 2: cài đặt maven (xem hướng dẫn trên mạng), 
          sau đó vào terminal
          1.cd vào project 
          2.cd server/com.hust.project
          3. chạy mvn spring-boot:run
        
-Client: 
    1.cd client
    2.npm install
    3.npm start
